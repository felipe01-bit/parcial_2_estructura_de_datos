## Archivos

### `pila.js` — Stack (LIFO)
- Implementa una **Pila** usando una clase en JavaScript.
- Permite agregar y eliminar objetos con propiedades como `Nombre` y `Edad`.
- Principio: **Last In, First Out** (el último en entrar es el primero en salir).

### `cola.js` — Queue (FIFO)
- Implementa una **Cola** usando una clase en JavaScript.
- Permite agregar y eliminar datos numéricos.
- Principio: **First In, First Out** (el primero en entrar es el primero en salir).

## Cómo ejecutar
__git bash__

node pila.js
node cola.js
